Projects

Ball is Life

This site was created as a final project for my Applied Internet Technology Class. It is a basic social media site for basketball fans, where users can post clips from youtube and comment on them. It also supports filtering by team.

Languages/Technologies used: JavaScript, Node.js, Express, MongoDB, HTML, CSS, SASS, Handlebars

Coat for Arms

This site was created at HackRPI 2014 with a group of friends. It gives users recommendations on what to wear, based on weather information at their location and preset preferences for their cold/heat tolerance.

Languages/Technologies used: JavaScript, Node.js, MongoDB, HTML, CSS

JSFizzle

MTG Booster Tutor

This iOS app was created for my iOS Programming class. It is a mini-game and utility app for fans of the popular trading card game Magic: the Gathering. Users can simulate opening packs of cards (one of the most entertaining aspects of a trading card game) without having to spend any money. It also simulates a "draft," and feautures a set of lists where users can view all the possible cards contained for each pack. Created with my friend Adam.

Languages/Technologies used: Objective-C, MTGJson API

Settlers of Catan
